# scanner-cmd

Keyboard Emulator 1.0.0

Keyboard Input Simulator

USAGE:
    scanner-cmd.exe [OPTIONS] --code `<code>`
```
OPTIONS:
    -c, --code <code>      Qr Code
    -d, --delay <delay>    Delay before input simulation start
    -h, --help             Print help information
    -V, --version          Print version information
```
